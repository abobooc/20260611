# 第一次追星就上手：演唱會任務挑戰 🎤

這是一個專為新手粉絲設計的闖關式互動教材網頁。透過模擬演唱會前的準備、理財規劃以及現場禮儀，讓使用者在遊戲中學習如何成為一名理性且優質的粉絲。

## 遊戲特色

- **闖關式體驗**：三個獨立關卡，必須依序完成才能解鎖。
- **互動性強**：包含背包整理（拖曳物品）、預算分配（槓桿調整）、情境判斷等設計。
- **教學導向**：提供詳細的正誤解說註解與結尾知識題庫。
- **進度保存**：使用 `localStorage` 自動記錄玩家的關卡進度與總分。

## 關卡介紹

1. **第一關：演唱會生存指南** - 學習物品準備與時間規劃。
2. **第二關：追星理財術** - 學習預算分配、區分必要支出與保留備用金。
3. **第三關：優質粉絲養成計畫** - 學習現場禮儀、網路禮儀與資訊判讀。

## 如何運行

1. 下載此專案所有檔案。
2. 使用瀏覽器開啟 `index.html` 即可開始挑戰。

## 技術說明

- 使用原生 HTML5, CSS3 (Grid/Flexbox), JavaScript 撰寫。
- 已配置 GitHub Actions 自動部署至 GitHub Pages。
